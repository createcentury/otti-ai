# mtg-followup — 議事録 → アクションアイテム展開

打ち合わせ後に議事録テキストを渡すと、
アクションアイテム・担当者・期限・次回アジェンダを自動生成します。

## 引数

$ARGUMENTS（議事録テキストを貼り付け、またはファイルパスを指定）

## 実行手順

1. **議事録の解析**
   以下を抽出してください:
   - 決定事項
   - アクションアイテム（担当者・期限付き）
   - 懸案事項・積み残し
   - 次回打ち合わせへの持ち越し議題

2. **アクションアイテムの整理**
   以下の形式で整理してください:
   ```
   - [ ] [タスク内容] @[担当者] 期限: [日付]
   ```

3. **Notion への保存（MCP）**
   Notion MCP を使って、プロジェクトの議事録ページに保存:
   - 決定事項
   - アクションアイテム一覧

4. **次回アジェンダの生成**
   - 積み残し事項をベースに次回アジェンダ案を生成
   - Google Calendar MCP で次回打ち合わせの予定を確認
   - アジェンダ案を Slack MCP で共有（確認ゲートあり）

5. **確認ゲート**
   Slack送信・Notion保存の前に内容を表示して確認を求めること。
