# daily-report — 日次レポート生成

当日の処理サマリと翌日のTODOを自動生成します。
毎朝9時のRalph Loopから自動実行されます。

## 引数

$ARGUMENTS（省略可: 対象日。省略時は当日）

## 実行手順

1. **当日のログ収集**
   `logs/YYYY-MM-DD/` 以下の全ログファイルを読み込んでください。

2. **CHANGELOG の参照**
   `CHANGELOG.md` から当日の実験・変更を取得してください。

3. **レポート生成**
   以下のセクションでレポートを生成:
   
   ```markdown
   # 日次レポート — YYYY-MM-DD
   
   ## 処理サマリ
   - 実行スキル: X件
   - 自動処理完了: X件
   - 人間確認に回した件数: X件
   - エラー: X件
   
   ## 自動化ROI
   - 推定削減時間: X時間X分
   
   ## 明日のTODO
   - [ ] ...
   
   ## 注意事項・フォローアップ
   - ...
   ```

4. **ファイル保存**
   `reports/YYYY-MM-DD-daily.md` に保存してください。

5. **Slack 通知**
   レポートのサマリをSlack MCP で送信（確認ゲートあり）。
