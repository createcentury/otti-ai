# CHANGELOG — 実験ノート

システムが学習・成長していく記録。
単なる変更履歴ではなく「何を試して何がわかったか」を記録する実験ノート。
Claude Code はセッションをまたいでこのファイルを参照し、コンテキストを維持する。

---

## フォーマット

```markdown
## [実験 #NNN] YYYY-MM-DD

### 背景・仮説
何を解決しようとしたか、なぜそのアプローチを選んだか

### 実施内容
- 具体的に変更・実行したこと

### 結果
- 成功: 何が達成できたか
- 失敗: 何がうまくいかなかったか（数値付きで）

### 学習・次のアクション
- 次に試すべきこと
- 繰り返さないべき失敗
```
