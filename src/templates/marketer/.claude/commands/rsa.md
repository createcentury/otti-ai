# rsa — RSA広告文の自動生成

レスポンシブ検索広告（RSA）用の広告コピーを生成します。
見出し15本（各30文字以内）と説明文4本（各90文字以内）を出力します。

## 引数

$ARGUMENTS（キャンペーンID、または広告ブリーフテキスト）

## 実行手順

1. **コンテキスト取得**
   - 引数がキャンペーンIDの場合: Google Ads MCP で現在の広告・パフォーマンスデータを取得
   - 引数がテキストの場合: そのままブリーフとして使用

2. **ブランドガイドライン確認**
   CLAUDE.md の「ブランドボイス・ガイドライン」を参照し、
   トーン・禁止ワード・文字数制限を確認してください。

3. **生成（サブエージェント分業）**
   - 見出し生成: 製品ベネフィット・機能・CTAのバリエーションを15本生成
     - 各30文字以内を厳守
     - ピン留め用（位置固定）の候補を3本明示
   - 説明文生成: 詳細ベネフィット・社会的証明・緊急性を4本生成
     - 各90文字以内を厳守

4. **自己検証**
   生成後、以下を必ず確認:
   - 全見出しが30文字以内か
   - 全説明文が90文字以内か
   - 禁止ワードが含まれていないか
   - ブランドボイスに沿っているか

5. **出力**
   Google Ads Editor でインポート可能なCSV形式で出力:
   `reports/YYYY-MM-DD-rsa-[campaign_id].csv`
   
   また、人間がレビューしやすい形式でも表示してください。

6. **確認ゲート**
   広告の公開・適用は確認ゲートを通すこと。
   承認後に Google Ads MCP で広告を更新してください。
